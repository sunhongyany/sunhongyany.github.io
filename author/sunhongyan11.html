    获取git仓库

    有两种取得git仓库的方法：

    1. 在现有目录中初始化仓库 git init

    2. 克隆现有的仓库 git clone [url]

    检查当前文件的状态

    git status

    跟踪新文件

    git add

    暂存已修改的文件

    git add

    git add 是个多功能命令，可以用它开始跟踪新文件，或者把已跟踪的文件放在暂存区，还能用于合并时

    把有冲突的文件标记为已解决状态。
