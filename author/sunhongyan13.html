  3. 常见的映射方法

    Published: 六 24 六月 2017

    常见的映射类型有dict，defaultdict，OrderDict。

    dict中有个setdefault方法，用于处理找不到的键。

        >>> d = {}
    >>> k = 'k'
    >>> d.setdefault(k, []).append('a')
    >>> d
    >>> {'k': ['a']}

    defaultdict处理找不到键的一个选择。

        >>> from collections import defaultdict

    >>> d = defaultdict(list)

    >>> d
    >>> defaultdict(list, {})

    >>> d['k'].append('a')

    >>> d
    >>> defaultdict(list, {'k': ['a']})

    >>> d['k']
    >>> ['a']
