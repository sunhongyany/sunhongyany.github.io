arguments参数

2017-09-06

arguments参数是传递给函数的所有参数的集合。这个集合有个名为length的属性，它包含了参数的个数。

var sum = function () {
    var i, total = 0;
    for (i = 0, i < arguments.length; i += 1) {
        total += arguments[i];
    }
    return total;
}
console.log(sum(1, 2, 3, 4));    // 打印出10

arguments类似数组，可以用下面的方法将其转化成数组：

var args = Array.prototype.slice.call(arguments);

函数作用域与块作用域

2017-09-05

JS变量以函数作为作用域。

function f() {
    function b() {
        console.log(i);
    }
    for (var i=0; i<10; i++) {
        bar();
    }
}

JS没有块作用域的概念，ES6引入了关键字let，可以用于生成传统的块作用域。

if (true) {
    let b = 2;   // b是该代码块的局部变量
    console.log(b);
}
console.log(b);    // ReferenceError
