js中各种调用方式

Publié le 07/09/2017 à 20:30
Par sunhongyan

Dans JavaScript.

  1. 作为函数调用

如果函数不是以方法，构造函数或通过apply(),call()调用，那么它只能作为一个函数被调用：

function add() {}
add();
var sub = function() {
};
sub();

当以这种模式调用时，this被绑定在全局对象上。

  1. 作为方法被调用

方法是作为对象属性的函数。对于方法来说，this被绑定在方法被调用时所在的对象上。

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>title</title>
<script>
function testF() { return this; }
console.log(testF());
var testFCopy = testF;
console.log(testFCopy());
var testObj = {
    testObjFunc: testF
};
console.log(testObj.testObjFunc());
</script>
</head>
<body>
</body>
</html>

前两次调用都是函数调用，因此this指向的是全局上下文，最后是方法调用，this看到的是函数上下文。

  1. 作为构造函数被调用

this会被绑定到新建的对象上。

var Person = function (name) {
    this.name = name;
}
Person.prototype.greet = function () {
    return this.name;
}
var albert = new Person('Albert');
console.log(albert.greet());

  1. arguments参数

    arguments参数是传递给函数的所有参数的集合。这个集合有个名为length的属性，它包含了参数的个数。

    var sum = function () {
    var i, total = 0;
    for (i = 0, i < arguments.length; i += 1) {
        total += arguments[i];
    }
    return total;
}
console.log(sum(1, 2, 3, 4));    // 打印出10

    arguments类似数组，可以用下面的方法将其转化成数组：
